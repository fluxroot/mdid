Build it
--------
Message Digest Integrity Database uses [Gradle][] as build system. To 
build it from source, use the following steps. 

### Get it
`git clone https://github.com/fluxroot/mdid.git`

### Build it
`./gradlew build`

[Gradle]: http://gradle.org
